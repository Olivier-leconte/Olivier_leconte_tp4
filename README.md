# Olivier_leconte_tp4


## Knight of the Montain


### Description du projet
Bonjour, aujourd'hui je vais essayer de faire un jeu de plateforme en 2d comme à la mario. Pour commencer, j'aime bien frieren l'animé, donc je vais mettre de la musique.
en deuxième, j'aimerais mettre les controles. Aller à droite, à gauche et sauter. Ensuite, créer une plateforme pour permettre de faire bouger le personnage. Bref, maintenant, 
je vais avoir beaucoup de mal à faire à faire le code pour bouger le personnage, mais je sens que je vais y arriver. C'était dur, mais j'y suis arrivé. Le design du va être vraiment simple pour les 3 niveaux.
Je vais essayer de faire un écran titre et un écran fin pour montrer que le jeu vient de commencer et de finir. J'avais aussi mis des pièces Bon semaine 2, maintenant que les bases sont fait, j'aimerais faire un genre de plus on avance dans le jeu plus le jeu et les mécaniques deviennent vraiment dure. Pour le premier niveau, je vais le laissser comme le tp2. Là je m'apprette à faire les portails du niveau 2. En gros, je vais faire un genre de truc comme
quoi quand le joueur va arriver dans le portail il va respawn dans le point du début incroyable non?! Bon c'était beaucoup plus dure de ce que je pensais, mais j'y suis arriver. Je vais commencer à faire le niveau 3 sachant que  je n'ai pas finit de design le niveau 2. Je vais aussi essayer de mettre les points tranchants dans les niveaux 2 et 3 pour permettre au joueur de ce faire eliminier des le début. Dernière semaine, j,ai pas trop eu le temps de travailler sur le projet, mais j'essaye de faire de mon mieux. Bon j'aia terminer le projet avec toute les idées que je voulais faire. je suis tellement fier de moi. Après toute ces nuits blanches. C'était vraiment cool.

## Documentation
```
┖╴Player
    ┠╴AnimatedSprite2D
    ┠╴CollisionShape2D
    ┠╴vie
    ┃  ┠╴TextureRect
    ┃  ┖╴Sprite4
    ┠╴Camera2D
    ┠╴jump_sfx
    ┖╴walking_sfx
```
L'animatedSprrite2D : sert à voire le personnage le skin que je lui ai mis.
CollisionShape2D : permet de mettre une collision sur le personnage afin qu'il interagisse avec le décort.
vie : permet de voir la vie du personnage
TextureRect : le rectangle permet de voir la vie.
Sprite4 : il s'agit du coeur du personnage (une image)
Camera2D: Une camera rapproché qui permet de suivre le personnage
jump_sfx : le song mit quand le personnage saute
walking_sfx: le song que fait le personnage quand il marche dans la map.

ce sont vraiment les prinipaux choses que j'ai pu utiliser pour le projet.
